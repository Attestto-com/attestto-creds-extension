# Attestto Creds

Self-sovereign identity wallet — browser extension for verifiable credentials.

## What it does

- Discover and present W3C Verifiable Credentials to websites
- Selective disclosure via SD-JWT — share only what's needed
- DIDComm v2 messaging
- Shamir key backup — split recovery across devices and guardians
- Compatible with [@attestto/id-wallet-adapter](https://github.com/Attestto-com/id-wallet-adapter) discovery protocol

## Status

In development. Not yet published to browser extension stores.
